## 🚀 Getting Started

### 1. Installation
To set up the required environment, ensure you have [Conda](https://docs.conda.io/en/latest/) installed, then run:

```bash
# Create the environment from the requirements file
conda env create -f environment.yml

# Activate the environment
conda activate your_env_name

```

### 2. Inference

Run inference using:
```python
    python3 ./inference.py /path/to/your/audio_or_dir /path/to/save/results.json
```
